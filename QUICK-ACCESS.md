Both services are running in background processes.

### Stop Services
Close the terminal windows or press Ctrl+C in each.

### Restart Backend
```bash
cd backend
venv\Scripts\activate
uvicorn app.main:app --reload --port 8000
```

### Restart Frontend
```bash
cd frontend
npm start
```

## Asset ID Format

`[Category][Country][Province][Company][Year][Sequence]`

Example: `MLALPBAVIS25015`
- M = Monitor
- LA = Lao PDR
- LP = Luang Prabang
- AVIS = Company
- 25 = 2025
- 015 = Sequence
